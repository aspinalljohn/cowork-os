# Cowork OS — start here (read automatically every session)

Cowork reads this file at the start of every session in this folder. It boots the operating system. **Do these two things first, before anything else:**

1. **Read `SKILL.md`** in this folder and follow it. That is the operating kernel: who I am, the folder map, the prime directives (plan before acting, route every deliverable to its subject, never destroy silently), my voice profile, and the playbook library.
2. **Read `memory.md`** in this folder and apply it. That is my standing facts, decisions, and corrections.

Then proceed with what I ask. When I invoke a playbook by name (for example "triage my inbox", "run research-brief on X", "run my weekly review"), open the matching file in `playbooks/` and follow it instead of improvising.

**Default to this folder, not connectors.** Workspace words mean files in this folder. In particular, **"inbox" means the `00-inbox/` folder here, never my email** — only use email/Gmail/Slack/web if I explicitly name the service. When unsure, ask before reaching for a connector.

---

This file is the only thing Cowork needs to find on its own. Everything else hangs off `SKILL.md`. To make this your own OS, change the titles *inside* `SKILL.md` and `memory.md` (for example "John OS"). Do **not** rename the files `CLAUDE.md`, `SKILL.md`, `memory.md`, `map.md`, or anything in `playbooks/` → they are read by their exact names.
