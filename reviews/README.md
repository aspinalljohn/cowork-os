# reviews

Daily and weekly reviews land here, dated: `reviews/daily-review-<YYYY-MM-DD>.md` and `reviews/weekly-review-<YYYY-MM-DD>.md`. Produced by the `daily-review` and `weekly-review` playbooks, on demand or on a schedule.

A named home, not a catch-all — reviews always write to the same predictable place, so scheduled runs are deterministic.
