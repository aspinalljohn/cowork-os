# 01-projects

Active work, **one subfolder per project**. Each project folder holds its own inputs, working files, **and its finished deliverables** — the work and its output stay together. When a project ships or goes dormant, move the whole folder to `99-archive/`.

A standalone one-off (a brief or memo not tied to existing work) gets its own folder here too — a project of one. Recurring streams have their own homes instead: content in `content/`, reviews in `reviews/`. There is no generic outputs bin.
